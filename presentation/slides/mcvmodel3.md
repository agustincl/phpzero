## MODELO MVC
### Model View Controler
7. **Dispatch**:
8. **Controlador**: se encarga de procesar las interacciones del usuario y realiza llamadas al modelo para obtener los datos que después pasará a la vista.
9. **Modelo**: representa la información con la que trabaja la aplicación.
10. **Vista**: transforma el modelo en una página web que permite al usuario interactuar con ella.
11. **Response**: respuesta obtenida a la petición. 
12. **Render**: Se maqueta la respuesta con el layout (patrón de diseño) y se muestra la vista final al usuario.
